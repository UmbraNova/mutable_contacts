The program asks user for the actions: add a contact or find a person in the list of contacts by last name.

Action "add contact": the program asks for the first and last name of the contact,
 then the phone number, adds them to the dictionary and displays the search dictionary of contacts.
If this person is already in the dictionary, a message is displayed.

Action "search for a person by last name": the program requests a last name and gives out all contacts
 with such last names and their phone numbers. Symbol search.
